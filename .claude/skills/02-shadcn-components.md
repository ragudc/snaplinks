# Skill: shadcn/ui Components — SnapLinks

## Componentes instalados
card, badge, skeleton, tooltip, select, input, button, separator,
switch, dialog, dropdown-menu, tabs, avatar, form, label, sonner

## Reglas
- Importar siempre desde `@/components/ui/`
- Usar `cn()` de `@/lib/utils` para clases condicionales
- Nunca modificar archivos en `src/components/ui/`
- Cada componente con datos async debe tener su Skeleton companion
- Usar `Sonner` (toast) para feedback de acciones del usuario
